* API Urls for course App 

    Method          API urls                           Operation 
                    Urls starts with base url 

    GET             /courses                           Get all course 

    GET             /courses/{courseId}                Get single course of
                                                       given id in url

    POST            /courses                           Add new course 

    PUT             /courses                           Update the course 

    DELETE          /courses/{courseId}                Delete the course Id




* src/main/java (SpringrestApplication) - Starting point of spring framework, classes will start from here.

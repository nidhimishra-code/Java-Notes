* CSS - Cascading Style Sheets	
  It's a language that describes the style of an HTML document 
  It's describes how HTML elements should be displayed on screen.

* SYNTAX 

     selector {

               property:value;
              
               }


    for ex - 
            
      #f {

            color:red;               [ since H1 have f as id]

         }


* <style></style> - we are making space for css styling by using "style" tag 
  style ke andar jo CSS likhenge wo body ke andar jitne bhi elements honge unpe apply hoga.


* We can apply CSS in three ways - 

  1. inline css - agar kisi particular element pe css lagate h to wo hota h inline css.

  Disadvantage of inline css - hme har element pe jake css lagana hoga.                                                          

2. internal css - head ke andar jo style laga ke css likhte h wo internal css hoga.

     Padding - Internal distance

   Disadvantage of internal css - can't apply changes on other file tags.

3. external css - using this css, we can apply styling in different files of project.


* Three type of selector - Tag, Class Id

* Tag Selector - Means with the help of tag, we can select multiple element.

* CLass Selector ki help se bhaut sare element ko apne according target kar sakte h .

* Id Selector - to target element uniquely 

* To watch more selector - https://www.w3schools.com/css/css_selectors.app

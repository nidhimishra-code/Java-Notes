* while vs do...while

   1. with a while loop, the condition to be evaluated is tested at the beginning of each loop iteration so if the conditional expression evaluates to false, the loop will never be executed.

   2. With do-while loop, the loop will execute at least once even if condition is false...


* for...in and for...of loops
    

* for...in loop 

  1. The for-in loop is a special type of a loop that iterates over the properties of an object, or the elements of an array.
  2. for-in loop gives key and value 


  syntax - 
 
          for(variable in object) 
          {
              // Code to be executed
          }


   * Used for iterating object and array.



* for...of loop 
  
 1. ES6 introduces a new for-of loop which allows us to iterate over arrays or other iterable objects (e.g. strings) very easily.
 2. for...of loop gives value.
 3. Object is not iterable.

  Syntax - 

          for(variable of iterableObject)

          }

* DOM - Document Object Model

  1. Platform and language independent model to represent the HTML or XML documents.
  2. It defines the logical structure of the documents and the way in which they can be accessed and manipulated by an application program.
  3. In the DOM, all parts of the document, such as elements, attributes, text, etc. are organized in a hierarchical tree-like structure;

In DOM terminology these individual parts of the document are known as nodes.

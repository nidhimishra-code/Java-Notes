* Suppose you have 200 test cases and 49 test failed, how would you collect the data of only failed test cases in testNG. Best way? 
 
ans. We will create one listner class which will capture all failed test data. OnTestFailure() runs only for failed tests. We will attach this with testNG and we can triger this particular test.

49 failed test cases will get stored in testeNG failed XML and we can run that XML.

           import org.testing.Resources;
           import org.testing.ArrayList;
           import org.testing.List;
      
           public class FailedTestListener implements ITestListener {
    
               public static List<String> failedTests = new ArrayList<>();
    
               @Override 
               public void onTestFailure(ITestResult result) {
                   failedTests.add(result.getName());
               }
            }
        
            onTestFailure() runs only for failed tests
            store: Test name
            (optionally class name, error, timestamp)
